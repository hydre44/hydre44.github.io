le métier de dévellopeur
tout d'abords le dévellopeur doit trouver un potentiel client puis organise un ou plusieurs entretien afin de comprendre ce que veut le client, après ces entretiens si le client demande un site web ont lui fera une proposition commerciale. Suite a cela on pourra commencer la réalisation du projet en commençant par faire une maquette du site ainsi que le cahier des charge ont pourra ensuite proposer le tout au client qui devra le validé et après cela ont pourra commencer le développement et quand celui si est terminer la mise en production peut commencer et s'en suis la période de recettage permettant au client de montrer les problèmes afin qu'ils soit modifier. la durée d'un projet peu varier selon sa complexité entre 1 semaines et plusieurs mois.
Les différents outils utilisés
Pour faire son travail le dévellopeur utilise beaucoup de logiciel diffèrent:
-Adobe XD qui permet de faire les maquettes d'un site
-playwind pour crée un code html et CSS ( l'esthétique du code)
-Strapi pour mettre des données dans le site
-visual studio code pour éditer le code
-GitHub pour héberger les codes sources
-clever-cloud pour mettre le site en ligne
